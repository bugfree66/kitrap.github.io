解释器

Python分几种:CPython,PyPy,Jython,IPython等.这些解析器实现方式各不相同,最常用的也是从官网下载下来的是CPython.
解析器运行有几种方式:

  • 交互式
    像Linux命令行一样,从终端获取输入,执行,输出.
  • 直接执行
    python -c cmd 直接执行命令.
    python -m module 直接运行指定的模块.

变量和命名规则

Python和其他的解释型语言一样,声明变量时无需指定类型,类型和值进行绑定.比如定义一个字典: d = {}.
变量分为全局变量和局部变量.在函数内部要使用全局变量需要使用global.
python命名规则和其他编程语言类似:

  • 标识符由字母,数字,下划线组成,不能以字母开头.
  • 不能使用语言默认的关键字作为标识符.

注释方法

  • 单行注释
    ‘#’右边到行结束之间的内容会被忽略
  • 多行注释
    ‘’’ … ‘’’三个单引号之间的内容被忽略.
    “”” … “””三个双引号之间的内容被忽略.
